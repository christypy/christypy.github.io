    📖 咖啡調製做法
    ☕ 美式/拿鐵基底：
    18/19/20g 粉 → 萃取 1:2-3g 時間大落在: 20-35秒
    ➔ 加糖版本：加3g糖粉

    ➔ 熱美式： 1.濃縮60g 2.熱水九分滿
    ➔ 大熱美式：1.濃縮80g 2.熱水九分滿
    ➔ 冰美式：1.冰塊180g 2.水130g 3.基底60g 4.冰塊補滿
    ➔ 氣泡冰美式：1. 60g濃縮加冰塊 2.用蒸氣打5秒 3.倒入整杯冰塊

    ➔ 外帶冰拿鐵：1.冰塊120g 2.濃縮80g 3.牛奶230g
    ➔ 內用冰拿鐵：1.冰塊120g 2.濃縮60g 3.牛奶160g
    ➔ 熱拿鐵：1.濃縮60g 2.牛奶230g打熱 3.融合
    淺培美式：
    靜 or 果 8.5 刻度
    🍵 抹茶歐蕾：
    粉 7g + 糖粉 6g + 熱水 60g 攪拌均勻，剩下用熱牛奶補滿。
    🍫 可可歐蕾：
    粉 7g + 糖粉 6g + 熱牛奶一些攪拌均勻，剩下用熱牛奶補滿。
    🍫 冰可可歐蕾：
    粉 10g + 糖粉 10g + 40度溫水50g攪拌均勻後，+冰牛奶250g
    冰塊補到9分滿。
    🔧 每日洗咖啡機流程
    • 卸下把手，將粉杯替換為無孔逆洗杯 。
    • 用一字起子轉開濾網螺絲，取下濾網清洗（小心燙，保管好螺絲）。
    • 把手加入一湯匙清潔粉並扣回。
    • 長按「1 + 5」按鈕啟動自動清洗。
    • 洗完將水倒掉，重複「1 + 5」清洗至無泡泡。
    • 卸下把手，鎖回螺絲與濾網，換回原本有孔粉杯。
    • 清水跑幾次觀察是否無泡沫
